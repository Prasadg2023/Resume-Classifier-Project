# 🚀 Resume Classifier Project

A Machine Learning + NLP based web application that classifies resumes into different job categories using Streamlit.

---

## 📌 Overview

This project analyzes resume text and predicts the most relevant job category using Natural Language Processing and Machine Learning techniques.

It helps automate resume screening and improves recruitment efficiency.

---

## 🔍 Features

- 📄 Input resume text  
- 🧹 Text preprocessing (cleaning, stopword removal)  
- 📊 TF-IDF vectorization  
- 🤖 Machine Learning classification model  
- 🌐 Interactive web app using Streamlit  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Scikit-learn  
- Pandas & NumPy  
- NLTK

## 🚀 How to Run Locally

1. Clone the repository:
git clone https://github.com/Prasadg2023/Resume-Classifier-Project.git

2. Navigate to the folder:
cd Resume-Classifier-Project

3. Install dependencies:
pip install -r requirements.txt

4. Run the app:
streamlit run streamlit_app.py
